Web simulations

Play around with AquaPlexus and Game of Life straight in your browser!

These simulations are powerful and light-weight JavaScript implementations of bulky Java predecessors.

Source code can be found in the download section along with the original programs in .jar and .exe format.

Scientific paper

This project started as a graduation project, so naturally there is a paper on the subject which explains the background, function, and applications of computer simulations.

Enclosed is a .pdf with the entire project, including infographics.

Downloads

All files associated with this project can be found here for download.

This includes .jar files for the simulations, java source code, .html, .css and .js source code, and infographics.
