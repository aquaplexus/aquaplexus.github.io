About AquaPlexus

AquaPlexus is a project by Mattias Ulmestrand and Andi Pärn, two students currently attending Hulebäcksgymnasiet near Gothenburg. The project started as a graduation project to which a Java program was written. The program is called AquaPlexus and is based on Craig Reynolds' Boids from 1986. Reynolds' program was extended to improve the accuracy of the simulation and adapt it to fish instead of birds. Below are some enhancement examples:

  1. Hydrodynamics: As water is much more viscous than air, fish experience strong drag forces that affect how they move. A formula for water resistance is embedded in the code and it can be observed that fish slow down in a lively manner as they stop swimming and leave schools.
  2. Spurting: Fish are known to make random spurts every now and then. The fish in the simulations will spurt at random about two times per second. This can be observed when lonely non-schooling fish suddenly make quick moves in random directions. This also happens in schools, but is much less noticeable.
  3. Chaos: Schools of virtual fish are excellent to conduct experiments on. The program was extended with a system to record 300 fish's positions and speed in two dimensions. Running the program with the same starting settings (and without random spurts) yields the same result after a set number of frames every time. By changing the position of a single fish just a little bit the end result is rather different. The structure of the flock is drastically changed, all because of chaos.
