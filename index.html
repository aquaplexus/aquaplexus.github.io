AquaPlexus home

Web simulations

Two simulation programs were written for this project to observe the chaotic nature of life itself: 'AquaPlexus', a continuation of Craig Reynold's 'Boids' and 'Game of Life', based on John Conway's Game of Life.

Unfortunately, these are bulky Java programs that require downloading to view them. Therefore two web-based versions were written in JavaScript to give easy access to these programs.

Scientific paper

This project started as a graduation project, so naturally there is a paper on the subject which explains the background, function, and applications of computer simulations.
